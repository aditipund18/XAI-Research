# Explainable Artificial Intelligence (XAI)

This repository contains my independent undergraduate research work on 
Explainable Artificial Intelligence (XAI).

## Description
This research is a conceptual and survey-based study focusing on:
- Black-box models and the need for transparency
- Model-specific and model-agnostic explainability
- Popular techniques such as LIME and SHAP
- Applications, challenges, and future research directions
